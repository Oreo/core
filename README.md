## Welcome
The [core] is a modified version of [meikoz's Basic core](https://github.com/meikoz/Basic),On the basis of which I wrote my own thoughts

## Usage Gradle
build.gradle中添加：
```
repositories {
    maven { url "https://dl.bintray.com/oreo/oreo/" }
}

dependencies {
    compile 'com.oreo:core:1.0.0'
}
```
